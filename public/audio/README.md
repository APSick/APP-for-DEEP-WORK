# Аудио для плеера

Большие файлы (>100 MB) не хранятся в репозитории.

- **Локально:** положи файлы `.mp3` / `.m4a` в эту папку; имена должны совпадать с путями в `src/components/MusicCard.tsx` (массив `TRACKS`).
- **Деплой (Vercel и др.):** на сервере этой папки с файлами нет. Варианты:
  1. Залей аудио на CDN (или хостинг с публичными URL) и в настройках сборки задай переменную окружения `VITE_AUDIO_BASE_URL` = базовый URL (например `https://твой-домен.com/audio`). Плеер будет запрашивать `VITE_AUDIO_BASE_URL + /audio/имя-файла.mp3`.
  2. Либо добавь файлы в `public/audio/` при сборке (скрипт/CI), чтобы они попали в билд.
